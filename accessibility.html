Přístupnost

Jak činíme náš web přístupný všem návštěvníkům

1. Příslib přístupnosti

Věříme na to, že web by měl být přístupný všem, bez ohledu na jejich fyzické schopnosti. Ing. Kristýna Dvořáčková je zavázána poskytovat webové stránky, které jsou přístupné lidem s postižením a seniorům v souladu se standardem WCAG 2.1 na úrovni AA.

2. Co jsme realizovali

Náš web má následující přístupnostní funkce:

  • Sémantický HTML: Používáme správnou strukturu HTML pro lepší interpretaci čtecím software
  • Alternativní texty: Všechny obrázky mají popis pro čtečky obrazovky
  • Navigace pomocí klávesnice: Web lze plně ovládat klávesnicí bez myši
  • Adekvátní kontrast barev: Text má dostatečný kontrast pro čitatelnost
  • Responsive design: Web se přizpůsobuje různým velikostem obrazovky a zařízení
  • Jasné navigační struktury: Logické pořadí a jednoduché ovládání
  • Jednoduché a srozumitelné jazyk: Text je napsán jasně a srozumitelně
  • Video s titulky: Videa obsahují titulky pro neslyšící diváky
  • Přenositelná velikost textu: Lze zvětšit či zmenšit dle potřeby

3. Technologie kompatibility

Náš web je kompatibilní s:

  • Čtečkami obrazovky (NVDA, JAWS, VoiceOver)
  • Zvětšovacími programy
  • Diktovacím software
  • Přepínači a jinými asistenčními zařízeními
  • Moderními webovými prohlížeči

4. Omezení a alternativy

I když se snažíme maximalizovat přístupnost, některé prvky mohou na určitých zařízeních fungovat lépe než na jiných. Pokud narazíte na problém, dejte nám prosím vědět. Pokud je to možné, poskytneme alternativu.

7. Klávesové zkratky

Zde jsou hlavní klávesové zkratky pro navigaci po webu:

  • Tab: Přesunutí na další prvek
  • Shift + Tab: Přesunutí na předchozí prvek
  • Enter: Aktivace odkazu nebo tlačítka
  • Escape: Zavření menu nebo modálního okna
  • Alt + U: Přesunutí na hlavní obsah (Skip navigation)

8. Standardy a certifikace

Náš web je navržen v souladu se standardy WCAG (Web Content Accessibility Guidelines) 2.1 na úrovni AA. Pravidelně provádíme testy přístupnosti a aktualizujeme web, abychom se vyhnuli problémům.

9. Stížnosti a zpětná vazba

Vaše zpětná vazba je pro nás cenná. Pokud najdete něco, co není přístupné, prosím kontaktujte nás. Budeme se snažit problém vyřešit co nejdříve.
